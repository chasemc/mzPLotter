
To install:
```
install.packages("remotes")
remotes::install_github("chasemc/mzPlotter")
```

To run:
```
mzPlotter::createReport(readPath = "FILE PATH TO MZML OR MZXML",
                        writePath = "FILE PATH OF NEW HTML")
```
